# Diabetes Health Dashboard

## Overview

An interactive Tableau dashboard analyzing the Diabetes Health Indicators dataset (BRFSS 2015), which contains over 253,000 survey responses. The dashboard compares health metrics, lifestyle factors, and outcomes across three groups: people with diabetes, prediabetes, and no diabetes.

---

## Key Dataset Variables

| Variable | Description |
|---|---|
| Diabetes_012 | Diabetes status (No Diabetes, Prediabetes, Diabetes) |
| HighBP / HighChol | High blood pressure / high cholesterol indicators |
| BMI | Body Mass Index |
| Smoker / Stroke | Lifestyle and medical history indicators |
| PhysActivity / Fruits / Veggies | Lifestyle behavior indicators |
| GenHlth / MentHlth / PhysHlth | Self-reported health status |
| Demographics | Sex, Age, Education, Income |

---

## Key Features

- **Interactive visualizations** — explore health metrics across diabetes status groups
- **Comparative analysis** — compare lifestyle, medical conditions, and outcomes by diabetes status
- **Demographic filters** — filter by age, sex, education, and income for deeper insights
